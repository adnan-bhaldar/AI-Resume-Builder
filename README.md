# ResumeDott – AI-Powered Resume Builder 🚀

ResumeDott is a modern web application designed to help users create professional, ATS-friendly resumes with the help of AI. It provides real-time resume scoring and intelligent content enhancement to improve job application success rates.

🌐 Live Demo: https://resumedott.vercel.app/

---

## ✨ Features

* 🧠 **AI Resume Enhancement**

  * Improves job descriptions using Google AI APIs
  * Suggests impactful, professional wording

* 📊 **ATS Score Checker**

  * Analyzes resume based on Applicant Tracking System standards
  * Provides actionable feedback to improve score

* 📝 **Dynamic Resume Builder**

  * Easy-to-use form-based input
  * Structured sections (Education, Experience, Skills, Projects)

* ⚡ **Real-Time Preview**

  * Instant visual updates while editing resume

* 📄 **Download Resume**

  * Export resume as a clean, professional PDF

* 🎯 **User-Friendly Interface**

  * Minimal and intuitive UI for smooth experience

---

## 🛠️ Tech Stack

* **Frontend:** React.js / Next.js
* **Styling:** Tailwind CSS / CSS Modules
* **Backend / APIs:** Google AI API
* **Deployment:** Vercel

---

## 🧠 How It Works

1. User fills in resume details
2. AI enhances descriptions using Google API
3. ATS algorithm evaluates resume content
4. Score and suggestions are generated
5. User downloads optimized resume

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/adnan-bhaldar/resumedott.git
cd resumedott
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env.local` file and add:

```env
GOOGLE_API_KEY=your_google_api_key_here
```

### 4. Run the Development Server

```bash
npm run dev
```

App will run on:

```
http://localhost:3000
```
